# kotoha — デジタルコンテンツ販売サイト

デザイン素材・デザインペーパー・ドット絵・結婚式アイテム（席次表/招待状）・オリジナルステッカーを扱う、**手づくりデジタル素材ショップ**のWebサイトです。

販売自体は **BASE / minne** で行い、このサイトは「作品カタログ＋ストアへの入口」として機能します。「いま何を作っているか」が一目でわかり、そのまま各ストアで購入できます。

## 構成

```
digital-shop/
├── index.html      # トップページ（ヒーロー / カテゴリー / おすすめ / 流れ / お知らせ）
├── shop.html       # 作品カタログ（カテゴリー絞り込み・BASE/minne購入ボタン）
├── about.html      # このお店について（ストーリー / こだわり / 利用規約）
├── contact.html    # お問い合わせフォーム（デモ）＋ よくある質問
├── css/style.css   # 全ページ共通スタイル（レスポンシブ対応）
├── js/main.js      # メニュー開閉 / スクロール表示 / 絞り込み / フォーム
├── img/            # ロゴ・カテゴリー別のSVGプレースホルダー画像
└── README.md
```

## 使い方

特別なビルドは不要です。`index.html` をブラウザで開くだけで表示できます。
GitHub Pages 等にそのまま公開することもできます。

## カスタマイズ方法

### 1. BASE / minne のリンクを設定する
各ページの購入ボタンや「販売ストア」リンクは現在 `href="#"`（仮）です。
実際のストアURLに置き換えてください。

- `shop.html` … `class="store-btn base"` / `class="store-btn minne"` の `href`
- 各ページ下部フッターの「販売ストア」リンク
- `index.html` ヒーロー下の BASE / minne リンク

### 2. 商品を追加・編集する
`shop.html` の `<article class="card" data-cat="...">` をコピーして増やせます。
`data-cat` の値（`material` / `paper` / `dot` / `wedding` / `sticker`）で
カテゴリー絞り込みが効きます。

### 3. 画像を差し替える
`img/` のSVGは仮の見本です。実際の商品写真（JPG/PNG）に差し替えてください。

### 4. ショップ名・色を変える
- ショップ名「kotoha」… 各HTMLと `img/logo.svg` を編集
- 配色 … `css/style.css` 冒頭の `:root` 変数（`--accent` など）を変更

## メモ
- お問い合わせフォームは**デモ動作**（実際の送信はしません）。Formspree 等の外部サービスや BASE/minne のメッセージ機能に繋ぐと実運用できます。
- 価格・商品名・お知らせ・規約はすべてサンプルです。実際の内容に書き換えてください。
